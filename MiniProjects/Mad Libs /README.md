# Mad Libs Game

## Description
A simple Java console project that generates a funny story based on user input.

## Concepts Used
- Variables
- Data Types
- User Input (`Scanner`)
- String Concatenation
- Output (`System.out.print`)
- Basic Program Structure

## How It Works
The program asks the user to enter:
- Name
- Subject
- Teacher Name
- Animal
- Number
- Verb

It then combines these inputs into a funny story.

## Sample Output

One day, Rajesh was sleeping in Maths class.Suddenly, Mr. Amit turned into a Monkey and startedDancing. Everyone laughed 10 times!
